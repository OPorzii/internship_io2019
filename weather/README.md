## วิธีใช้งาน
1. ติดตั้ง Python Module เปิด command พิมพ์
```
pip install xmltodict
```

2. เปิดไฟล์ xml_convert.py และใส่ path ของไฟล์ xml ที่ต้องการจะแปลงเป็น json
3. โปรแกรมจะขึ้นข้อความว่า
```
Convert Success!!
```
